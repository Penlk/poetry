# ТЗ на стихи

## Отрабатываемые материал
Вы)

## Цель
Сделать мне приятно при помощи вашего интеллектуального труда

## Формулировка
Вы придумываете прикольное стихотворение про меня. Которое должно мне понравится)
Впоследствии выкладываете результат ваших работ в чат

После того, как все прочтут свои стихи я выбираю одного победителя.

## Функциональные требования
Необходимо поставить звезду этому репозиторию))

**count_row** (int) - количество строк в стихе

**count_words** (int) - количество слов в стихе

**poetry** (```std::vector<std::string>```) - стих в строках

```
int counter_words = 0;
for (int i = 0; i < poetry.size(); ++i) {
	std::istringstream iss(poetry);
	std::string word;
	
	while(iss >> word) {
		++counter_words;
	}
}
```

Тогда poetry должен выполнять следующее условие:
(poetry.size() >= 4 && poetry.size() <= 6) && (counter_words >= 30 && counter_words <= 50) 

Так же необходимо выложить ваше стихотворения в чат.

## Нефункциональные требования
Стих на тему меня: можно мне польстить, можно немного высмеять, можно выразить уважение ко мне.

Желательные поэты: Маяковский, Блок

Для Алексанра: Оксимирон
